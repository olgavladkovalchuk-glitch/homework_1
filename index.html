Практическое задание

Создай HTML-страницу в Блокноте

Потренируйся в песочнице → проверь по чеклисту → сдай домашку

📋 Задание

Создай HTML-страницу в Блокноте со следующим содержанием:

  1. Каркас: <!DOCTYPE html>, <html>, <head>, <body>
  2. Кодировка <meta charset="UTF-8"> и <title>
  3. Фон и цвет текста — атрибуты bgcolor и text у <body>
  4. Заголовок <h1>, два абзаца <p>, список <ul> или <ol>
  5. Ссылка <a href="..."> на любой сайт

Сохрани файл как index.html и отправь в папку Домашки

⏰ Дедлайн: 13 марта, 23:59 МСК
Как сохранить файл
  1. Открой Блокнот (Пуск → набери «Блокнот»)
  2. Напиши или скопируй код
  3. Файл → Сохранить как… → имя index.html → тип Все файлы → кодировка UTF-8
  4. Открой двойным кликом — проверь в браузере
